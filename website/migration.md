# Migration Guide

## Current Version

MeowKit is at version 1.0.0. This is the initial release.

## Future Migrations

When new versions are released, migration guides will appear here with:

- Breaking changes
- Deprecated features  
- Step-by-step migration instructions
- Compatibility notes

## Upgrading

```bash
npx mewkit upgrade
```

This command handles most migrations automatically.
