# mk:ui-design-system

Data-driven design intelligence: curated palettes, font pairings, product recommendations, UX guidelines, accessibility standards, and quality checklists. Auto-activates when `ui-ux-designer` agent is working.

## When to use

- Creating or reviewing UI components
- Setting up a design system
- Checking WCAG accessibility compliance
- Selecting color palettes, typography, or UI styles
- Choosing chart types for data visualization

## Assets

160 color palettes, 73 font pairings, 161 product types, 98 UX guidelines, 25 chart types. Data-driven from CSV reference files.
