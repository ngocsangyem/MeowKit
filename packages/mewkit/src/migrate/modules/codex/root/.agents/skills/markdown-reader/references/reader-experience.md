# Markdown Reader Experience

The reader provides light/dark themes, a comfortable 720px text column, reading progress, syntax
highlighting, directory browsing, and plan-directory navigation. It recognizes `plan.md` plus
`phase-*.md`, adds status badges and previous/next controls, and uses a mobile sheet on small
screens.

Keyboard shortcuts: `?` help, `T` theme, `S` sidebar, arrow keys phase navigation, and `Esc`
close. Mermaid fenced blocks render theme-aware diagrams; errors show the source for correction.
Supported types include flowchart, sequence, pie, gantt, xychart, mindmap, quadrant, ER, state,
and timeline.
